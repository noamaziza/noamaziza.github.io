4 GAMES IN 1 by Noam

Le Memory

Règle:


Toutes les cartes sont étalées. L'objectif est de trouver toutes les paires avec le moins de coup possible.La partie se termine quand toutes les cartes sont découvertes.


La Machine à sous

Règle:


Si vous souhaitez remporter le Jackpot ? Obtener un doublé ou un triplé du même symbole et rempoter la mise. Le jeu s'arrête quand vous n'avez plus rien à miser.




Le Rapido

Règle:


Le but est de trouver 8 nombres parmi 20 dans une première grille (grille A) et simultanément un chiffre parmi quatre (grille B).Le jeu s'arrête quand vous n'avez plus rien à miser.
